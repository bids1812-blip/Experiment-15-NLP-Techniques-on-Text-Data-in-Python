# README: NLP Techniques on Text Data in Python

## Aim
To implement and understand basic Natural Language Processing (NLP) techniques using Python, including tokenization, stop word removal, stemming, lemmatization, POS tagging, and word frequency analysis.

## Theory
Natural Language Processing (NLP) is a branch of artificial intelligence that enables computers to understand, interpret, and process human language.

This experiment covers the following core NLP techniques:

1. Tokenization
- Splitting text into smaller units like words or sentences.
(Word Tokenization & Sentence Tokenization)
2. Stop Word Removal
- Eliminating commonly used words (like is, the, in) that do not add significant meaning to the text.
3. Stemming
- Reducing words to their root/base form by removing suffixes.
- Example: running → run
4. Lemmatization
- Converting words into their meaningful base (dictionary) form.
- Example: better → good
5. Part-of-Speech (POS) Tagging
- Assigning grammatical roles (noun, verb, adjective, etc.) to words.
6. Word Frequency Count
- Counting how often each word appears in a text.

These techniques are fundamental for preprocessing text data before applying machine learning or deep learning models.
